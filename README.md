# Lafz Welfare Foundation

Lafz Welfare Foundation is a social welfare organization dedicated to education, health, security, awareness, and environmental causes in the Sihora region.

## Project Optimization

This website has been refactored and optimized for better performance, modern UI/UX, and mobile responsiveness.

### Key Features:

- **Clean Architecture**: Separated CSS and assets for better maintainability.
- **Premium Design**: Modern color palette and typography (Poppins & Noto Sans Devanagari).
- **Fully Responsive**: Optimized for all devices.
- **SEO Ready**: Optimized meta tags and semantic HTML.
